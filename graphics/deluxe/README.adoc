= Deluxe skin source art

Redrawn full-color art for the *Deluxe* graphics skin
(Options > Graphics > Deluxe). See `docs/graphics-roadmap.adoc` for the
plan and the remaining redraw workload.

The `gen/` subdirectory is generated by
`micropolisj.build_tool.GenerateDeluxeArt` during `build-tiles.sh` and
is gitignored — don't put hand-made art there. Committed sheets in this
directory always override the generated ones.

How it works:

* Drop a redrawn sheet here as `name_64x64.png` — 64px-per-tile basis,
  same sheet layout and row order as the base `graphics/name.png`, so
  `tiles.rc` and the `.ani` files work unchanged.
* `build-tiles.sh` stages the modern sources, overlays the files in this
  directory, and composes `src/main/resources/deluxe/{8x8,16x16,32x32,64x64}/`.
  The 32/16/8px variants are derived from the 64px redraw by box-filter
  halving; commit an explicit `name_32x32.png` / `name_16x16.png` /
  `name_8x8.png` here only when the automatic downscale isn't good enough.
* Sheets without a redraw in this directory simply fall back to the
  modern art, so the skin stays usable while the redraw is incomplete.
* Keep tile color averages close to the original palette where redrawn
  tiles meet still-upscaled neighbours, and keep park trees clearly
  darker than forest trees at every size.
